PROJECTS

Multi tool

This batch tool is a simple command-line utility for Windows that provides a basic interactive menu. When run, it displays a welcome message and presents the user with three options: open Notepad, open Calculator, or exit the program. Based on the user's input, it launches the selected application or exits the tool. The script runs in a loop, allowing the user to perform multiple actions until they choose to exit. It uses a black background with green text for a clean, hacker-style aesthetic and is ideal for basic task automation or learning batch scripting.

Bruteforce Tool

This batch tool is a simple command-line utility that provides quick access to basic system information through a menu interface. When run, it displays four options: viewing IP configuration, checking system information, listing the contents of the current directory, or exiting the tool. The user selects an option by entering a number, and the corresponding command is executed. After displaying the result, the tool pauses to let the user review the output and then returns to the main menu. It features a colored interface for better readability.
